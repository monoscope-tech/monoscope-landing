This week introduced major onboarding improvements and a new API testing feature.

## New Features

### Onboarding Pages

Completely redesigned onboarding experience with dedicated pages for each setup step, making it easier for new users to get started with Monoscope.

### API Test Page

New API testing interface allowing users to:

- Send test requests to verify SDK integration
- Validate that data is being captured correctly
- Debug integration issues before going to production

## Improvements

### UI Design Refresh

- Redesigned input fields, select dropdowns, and checkboxes
- Updated side navigation with new active state styling
- Improved visual consistency across the platform
