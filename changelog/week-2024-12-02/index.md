This week marks a major milestone with the launch of our custom dashboard system.

## New Features

### Custom Dashboards

Introduced a full-featured dashboard system allowing you to:

- Create, edit, and delete custom dashboards
- Build personalized views of your API metrics
- Organize widgets to match your workflow

### Widgets and Charts

- Chart list widget for visualizing time-series data
- Graph rendering with multiple visualization options
- Widget count displayed on dashboard list for quick overview

### Data References

Added "Referenced In" column to track where data is being used across your dashboards.

## Improvements

### Navigation

- Sidenav improvements with better collapse/expand behavior
- Progress toward full dashboard navigation integration

## Bug Fixes

- Fixed lift error in dashboard rendering
- Disabled API test emails and daily report emails (temporarily)
